# Levantamento de Materiais

O módulo de **Levantamento de Materiais** analisa automaticamente os pedidos de marcenaria e gera a lista de materiais necessários para produção.

---

## Como funciona

1. Importe o pedido (PDF ou planilha)
2. O sistema processa os itens automaticamente
3. É gerado um relatório com quantitativos de chapas, ferragens e insumos
