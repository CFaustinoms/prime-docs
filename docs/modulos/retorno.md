# Retorno de Obra

O módulo de **Retorno de Obra** controla materiais, equipamentos e sobras que retornam da instalação.

---

## Etiquetas QR

Cada item de retorno recebe uma etiqueta QR Code impressa. Ao chegar na fábrica, o operador escaneia a etiqueta para registrar a entrada.

---

## Dashboard de retornos

O dashboard exibe todos os retornos pendentes, permitindo acompanhar o que ainda está em obra e o que já voltou para a fábrica.
