# Frota

O módulo de **Frota** controla os veículos da Prime — abastecimentos, manutenções e uso.

---

## Dashboard de frota

Visão geral de todos os veículos com:
- Quilometragem atual
- Último abastecimento
- Manutenções pendentes

---

## Controle mobile

O módulo tem uma tela mobile otimizada para que o motorista registre abastecimentos e ocorrências diretamente pelo celular, sem precisar de computador.
