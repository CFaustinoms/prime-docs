# Gestão de Usuários

O módulo de **Administração > Usuários** permite criar, editar e inativar os usuários do sistema.

---

## Criar usuário

1. Acesse **Admin → Usuários**
2. Clique em **Novo Usuário**
3. Preencha nome, login, perfil e vincule ao colaborador correspondente
4. O usuário receberá a senha padrão `prime@2026` e deverá trocá-la no primeiro acesso

---

## Login padrão

O login segue o formato `nome.sobrenome`. Ex: `carlos.faustino`

---

## Inativar usuário

Usuários inativos não conseguem fazer login, mas seus dados e histórico são preservados no sistema.

---

## Vincular ao colaborador

Todo usuário deve ser vinculado ao seu registro de colaborador. Isso permite que o sistema mostre informações personalizadas em **Meu Perfil**.
