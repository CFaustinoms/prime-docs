# Custo Operacional

O módulo de **Custo Operacional** calcula o custo real de cada colaborador CLT, incluindo todos os encargos trabalhistas.

---

## Encargos calculados

| Encargo | Percentual |
|---|---|
| FGTS | 20% |
| INSS patronal | 20% |
| RAT | 2% |
| Sistema S | 5,8% |
| **Total encargos** | **47,8%** |

Além dos encargos, o sistema considera provisões de 13º salário, férias e 1/3 de férias.

---

## Custo por hora

Com base no salário e encargos, o sistema calcula o **custo por hora** de cada colaborador — usado nos relatórios de custo de produção por OP.

---

:::caution Acesso restrito
Este módulo é visível apenas para perfis com acesso liberado pelo administrador.
:::
