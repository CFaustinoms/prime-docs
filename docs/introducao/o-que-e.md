# O que é o Prime Gestão de Produção?

O **Prime Gestão de Produção** é o sistema interno da Prime Marcenaria Técnica para controlar toda a operação — do pedido aprovado até a entrega na obra.

Ele substitui planilhas Excel, Trello e grupos de WhatsApp por uma única plataforma web, acessível de qualquer computador ou celular.

---

## O que você encontra no sistema

| Área | O que faz |
|---|---|
| **Pedidos Gerais** | Visão de todas as ordens de produção em andamento |
| **Projetos** | Atribuição de projetista e acompanhamento do status do projeto |
| **Produção / PCP** | Programação, prioridades e controle de processos por setor |
| **Apontamento** | Registro de horas trabalhadas por processo via QR Code |
| **Expedição** | Conferência de volumes e emissão de romaneios |
| **Almoxarifado** | Controle de equipamentos, insumos e empréstimos |
| **Colaboradores** | Ficha completa, ocorrências, documentos e férias |
| **Frota** | Controle de veículos e abastecimentos |
| **Administração** | Usuários, permissões e auditoria do sistema |
