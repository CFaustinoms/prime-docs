# Perfis de acesso

Cada usuário do sistema possui um **perfil** que define quais módulos ele pode acessar e o que pode fazer dentro de cada um.

---

## Níveis de acesso

| Nível | O que permite |
|---|---|
| **Total** | Ver, criar, editar e excluir |
| **Edição** | Ver, criar e editar (sem excluir) |
| **Leitura** | Somente visualizar |
| **Sem acesso** | Módulo não aparece no menu |

---

## Perfis disponíveis

| Perfil | Descrição |
|---|---|
| **ADMIN** | Acesso total ao sistema |
| **GESTOR_SISTEMA** | Gestão técnica do sistema |
| **GESTOR** | Gestão operacional geral |
| **DIRETORIA** | Visão gerencial e financeira |
| **PCP** | Programação e controle da produção |
| **PROJETISTA** | Projetos e liberação para produção |
| **MARCENARIA** | Apontamento e processos de produção |
| **ALMOXARIFADO** | Controle de equipamentos e insumos |
| **LOGISTICA** | Expedição e frota |
| **INSTALACAO** | Checklist e retorno de obra |
| **RH** | Colaboradores e documentos |
| **ADMINISTRATIVO** | Visão administrativa geral |
| **COMPRAS** | Levantamento de materiais |
| **LABORATORIO** | Processos de laboratório |
| **LUSTRACAO** | Processos de lustração |
| **MANUTENCAO** | Diário de equipamentos |
| **ORCAMENTOS** | Orçamentos e pedidos |
| **OPERADOR** | Apontamento via QR Code |

---

:::info
Além do perfil, o administrador pode configurar **permissões individuais** para um usuário específico — liberando ou bloqueando módulos independentemente do perfil geral.
:::
